Privacy Policy – Below Emerald Reader

Effective Date: July 2025

1. Who We Are

This project is operated by Mads S. Balto for the purpose of anonymous message analysis within the MLA Discord server. Questions or concerns can be directed to:

  • Mads S. Balto
  • https://www.madsbalto.com

2. What Data We Collect

The Below Emerald Reader bot collects and stores the following data from consenting Discord users:

  • Encrypted message content (AES-256)
  • Hashed user ID (SHA-256)
  • Highest self-assigned rank (as a text label)
  • Deduplication hash for record integrity

No usernames, plain user IDs, or message IDs are stored. Messages are collected only from users who explicitly opt in via the /consent command.

3. Why We Collect Data

Collected data is used solely to analyze message trends by rank and topic for educational insights and coaching improvement. Data is not shared, sold, or used for any other purpose.

4. How Data Is Stored and Protected

All data is encrypted and stored securely in a controlled database. The encryption key is kept separately in an environment-secured configuration file. Reasonable technical and organizational measures are taken to protect data confidentiality.

5. Your Rights

At any time, you may:

  • Withdraw your consent via /consent
  • Request deletion of your stored data
  • Request access to your stored messages

All data is deleted automatically upon project conclusion.

6. Legal Basis

Processing is based on freely given, explicit consent under GDPR Article 6(1)(a).

7. Data Retention

Data is retained only for the duration of the study. Upon completion or upon user withdrawal, all associated data is deleted securely.

8. Changes to This Policy

This Privacy Policy may be updated at any time. Users will be notified via the bot if significant changes affect data usage or user rights.
